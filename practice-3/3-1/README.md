Задание 3-1: Неименованные каналы pipe()<br>
Требования:<br>
• Используйте pipe(), fork(), read(), write(), close().<br>
• Не используйте высокоуровневый ввод-вывод fopen(), fprintf(), fscanf().<br>
• Обрабатывайте ошибки через perror().<br>
• Компиляция: gcc -Wall -Wextra -o task task.c.<br>

Вариант 6. Родитель читает строку с клавиатуры и передаёт её дочернему через pipe().<br> 
Дочерний переводит все строчные буквы в заглавные и выводит результат.<br>

Используемые системные вызовы:
pipe() - создает неименованный канал (pipe), возвращает два файловых дескриптора: pipefd[0] (чтение) и pipefd[1] (запись)<br>
fork() - создает дочерний процесс<br>
read() - низкоуровневое чтение данных из файлового дескриптора<br>
write() - низкоуровневая запись данных в файловый дескриптор<br>
close() - закрывает файловый дескриптор<br>
wait() - ожидает завершения дочернего процесса<br>

Алгоритм работы:<br>
Родительский процесс:<br>
Читает строку с клавиатуры через fgets()<br>
Записывает строку в pipe через write()<br>
Закрывает свой конец pipe<br>
Ожидает завершения дочернего процесса через wait()<br>

Дочерний процесс:<br>
Читает строку из pipe через read()<br>
Преобразует все строчные буквы в заглавные с помощью islower() и toupper()<br>
Выводит результат на экран<br>
Закрывает свой конец pipe<br>
