Задание 3-2: Именованные каналы FIFO<br>
Требования:<br>
• Используйте mkfifo(), open(), read(), write(), close(), unlink().<br>
• Обрабатывайте ошибки.<br>
• Компиляция: gcc -Wall -Wextra -o task task.c.<br>

Вариант 6. Один процесс каждую секунду отправляет текущее время в FIFO,<br>
второй принимает и записывает в time.log.<br>

Используемые функции:<br>
mkfifo() - создает именованный канал (FIFO) в файловой системе<br>
open() - открывает для чтения или записи<br>
read() - чтение данныx<br>
write()	- запись данных<br>
close()	- закрывает файловый дескриптор<br>
unlink() - удаляет FIFO из файловой системы<br>
fork()	- создает дочерний процесс<br>
wait()	- ожидает завершения дочернего процесса<br>
time()/localtime()/strftime()	- получает и форматирует время<br>

Алгоритм работы программы:<br>
Инициализация main():<br>
Удаляет FIFO если он существует unlink()<br>
Создает именованный канал mkfifo()<br>
Создает дочерний процесс fork()<br>

Дочерний процесс:<br>
Открывает FIFO для чтения open()<br>
Открывает/создает лог-файл time.log для записи<br>
В бесконечном цикле читает данные из FIFO<br>
Удаляет символ новой строки из полученных данных<br>
Записывает время в лог-файл<br>
При ошибке или закрытии FIFO завершает работу<br>

Родительский процесс:<br>
Ждет 1 секунду (дает получателю время на открытие FIFO)<br>
Открывает FIFO для записи open()<br>
В цикле из 10 итераций:<br>
Получает текущее время<br>
Форматирует его в строку<br>
Записывает время в FIFO<br>
Ждет 1 секунду sleep(1)<br>
Закрывает FIFO и завершается<br>

Родитель ждет завершения дочернего процесса wait()<br>
Удаляет FIFO из файловой системы unlink()<br>
