Задание 4-1: Динамическая память<br>
Требования:<br>
• Используйте malloc(), calloc(), realloc(), free().<br>
• Проверяйте результат выделения памяти.<br>
• Компиляция: gcc -Wall -Wextra -o task task.c. <br>

Вариант 6. Реализовать динамический список строк, вводимых пользователем, <br>
затем вывести их в обратном порядке.<br>

Используемые функции:<br>
malloc() - выделяет блок памяти заданного размера в куче (heap)<br>
realloc() - изменяет размер ранее выделенного блока памяти<br>
free() - освобождает ранее выделенную память<br>
fgets() - читает строку из стандартного ввода<br>
strlen() - возвращает длину строки<br>
strcpy() - копирует строку из одного места в другое<br>

Алгоритм работы программы:<br>
Основной цикл ввода в бесконечном цикле:<br>
Выводит приглашение к вводу (printf)<br>
Читает строку через fgets() в буфер<br>
Удаляет символ перевода строки (\n) из буфера<br>
Проверяет условие завершения: если строка пустая (длина 0) - выход из цикла<br>
Вычисляет размер строки: (len + 1) * sizeof(char)<br>
Вызывает malloc() для выделения памяти под новую строку<br>
Проверяет результат: если NULL - выводит ошибку и освобождает ранее выделенную память<br>
Копирует содержимое буфера в выделенную область с помощью strcpy()<br>
Вызывает realloc() для увеличения размера массива lines на один элемент<br>
Проверяет результат: если NULL - обрабатывает ошибку<br>
Сохраняет указатель на новую строку в массиве<br>
Увеличивает счетчик count<br>

После завершения ввода:<br>
Выводит заголовок "Строки в обратном порядке:"<br>
В цикле от count - 1 до 0 Выводит строку с номером<br>
В цикле от 0 до count - 1 вызывает free() для каждой строки<br>
Вызывает free() для массива указателей lines<br>
