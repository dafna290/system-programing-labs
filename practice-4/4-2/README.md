Задание 4-2: Отображение файлов в память mmap()<br>
Требования:<br>
• Используйте open(), fstat(), mmap(), munmap(), close().<br>
• Обрабатывайте ошибки.<br>
• Компиляция: gcc -Wall -Wextra -o task task.c.<br>

Вариант 6. Отобразить двоичный файл с массивом целых чисел и вычислить среднее значение.<br>

Используемые функции:<br>
open() - открывает файл<br>
close() - закрывает файл<br>
fstat() - получает информацию о файле по его дескриптору<br>
mmap() - отображает файл в память процесса<br>
munmap() - удаляет отображение файла из памяти процесса<br>

Алгоритм работы программы:<br>
Проверяет аргументы командной строки<br>
Открывает файл<br>
Получение информации о файле<br>
Проверка корректности файла<br>
Вычисление количества элементов<br>
Отображает весь файл в память mmap()<br>
Закрытие файлового дескриптора close(fd) <br>
Вывод и суммирование чисел<br>
Вычисление среднего значения<br>
Вывод результатов количество элементов и среднее значение <br>
Освобождает память, занятую отображением munmap()<br>
